# Project Plan

This file tracks the completed milestones for turning the original PDF conversion script into an npm library.

## Completed Milestones

- [x] Extract the PDF to Markdown/DOCX pipeline into a reusable TypeScript library.
- [x] Expose the converter through a Node CLI and npm package exports.
- [x] Document installation, CLI usage, library usage, scan-specific behavior, and test fixtures in English.
- [x] Remove Docker and Make-specific workflows so the project stays a standard npm package.
- [x] Add npm-based CI for build/type-check and package validation.
